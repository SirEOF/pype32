What's pype32?
======

Yet another Python library to read and write [PE/PE+](http://es.wikipedia.org/wiki/Portable_Executable) files.

Usage
======

```python
>>> import pype32
>>> pe = pype32.PE("C:\Windows\notepad.exe")
```
